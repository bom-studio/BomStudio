Place WebP screenshots for each service type:

- `corporate.webp` — 기업 홈페이지
- `hospital.webp` — 병원
- `academy.webp` — 학원
- `cafe.webp` — 카페·음식점
- `shop.webp` — 쇼핑몰
- `landing.webp` — 랜딩페이지
- `booking.webp` — 예약 시스템
- `admin.webp` — 관리자 페이지

Recommended: 16:10 aspect ratio, ~1280px wide.

Until images are added, icon-based placeholders are shown automatically.
